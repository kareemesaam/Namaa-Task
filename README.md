# Namaa

## Installation

First clone this repository, install the dependencies
```
git clone https://github.com/kareemesaam/Namaa-Task.git
composer install
cp .env.example .env
php artisan key:generate

```

API That gets users 

```
users EndPoint 
GET : 
http://127.0.0.1:8000/api/v1/users
```

Run Test 
```
php artisan test
```
